[前置設定－環境變數]
建立.env檔案，並參考.env.example進行設置。

設定值說明:
TOKEN=機器人金鑰
TZ=時區 (以便後台紀錄，例:台灣為8，紐約為-4)
DB_HOST=Redis資料庫連線地址
DB_PORT=Redis資料庫連線端口
DB_USERNAME=Redis資料庫登入使用者名稱
DB_PASSWORD=Redis資料庫登入密碼
OWNER_ID=機器人擁有者ID

[功能說明－頻道訊息搬移]
# 轉傳監聽頻道的訊息與圖片到指定頻道，並儲存於messageLog.xlsx。有管理UI設定訊息發送時tag的身分組、監聽頻道ID、轉傳頻道ID。
- 監聽頻道所需權限: 檢視頻道
- 轉傳頻道所需權限: 發送訊息
- 管理UI指令: /ascchannel
- 管理UI使用權限: 機器人擁有者

[功能說明－DISCORD社群字典]
# 查詢特定關鍵字的解釋。
- 指令: /ascgoogle <關鍵字>

[功能說明－檢視連結清單]
# 發送指令出現注意事項，點選同意後，出現UI顯示數個標題超連結。
- 指令: /asclink

[功能說明－授權系統]
# 邀請機器人後，透過使用授權密碼解鎖功能。
- 產生授權密碼指令: /ascpass
- 指令使用權限: 機器人擁有者

[功能說明－公告系統]
# 發布公告於指令頻道中。
- 指令: /ascpost <頻道>
- 指令使用權限: 伺服器管理員

[功能說明－使用者驗證]
# 驗證社群成員是否在線活躍或是掛機
- 指令: /asccheck
- 指令使用權限: 伺服器管理員

[功能說明－智慧王]
# 生成題目及選項，最快答對者獲勝。
- 指令: /ascknow

[文件說明－ascgoogle.xlsx]
A欄: 關鍵字
B欄: 解釋

[文件說明－asclink_terms.txt]
/asclink注意事項

[文件說明－asclink.xlsx]
A欄: 標題
B欄: 連結
C欄: 描述
D欄: 圖片連結

[文件說明－ascknow.xlsx]
A欄: 題目
B欄: 答案序號
C~Z欄: 選項

[文件說明－messageLog.xlsx]
A欄: 設定檔名稱
B欄: Unix時間戳
C欄: 時間戳
D欄: 使用者名稱:使用者ID
E欄: 原始訊息ID
F欄: 轉傳訊息ID
G欄: 訊息內文
H欄: 附加檔案連結, 附加檔案連結, ...
I欄: 原始訊息頻道ID
